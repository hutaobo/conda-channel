# hutaobo conda channel

Static conda channel for selected hutaobo Python packages while conda-forge
feedstocks are under review.

Install with:

```bash
conda install -c https://raw.githubusercontent.com/hutaobo/conda-channel/main -c conda-forge cell-gps pyxenium histoseg
```

Packages:

- `cell-gps 0.0.4`
- `pyxenium 0.4.6`
- `histoseg 0.1.9.3`

The conda-forge staged-recipes PRs remain the canonical path for long-term
distribution.
